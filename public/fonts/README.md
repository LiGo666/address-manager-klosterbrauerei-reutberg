# Fonts

Die folgenden Font-Dateien müssen hier abgelegt werden:

## Libre Baskerville
- `libre-baskerville-regular.woff2` (weight: 400)
- `libre-baskerville-regular.woff` (weight: 400, Fallback)
- `libre-baskerville-bold.woff2` (weight: 700)
- `libre-baskerville-bold.woff` (weight: 700, Fallback)

## Fonts herunterladen

Die Fonts können von Google Fonts heruntergeladen werden:
- https://fonts.google.com/specimen/Libre+Baskerville
- https://fonts.google.com/specimen/IBM+Plex+Mono
- https://fonts.google.com/specimen/Lora

Oder mit einem Tool wie `google-webfonts-helper`:
- https://gwfh.mranftl.com/fonts/libre-baskerville
- https://gwfh.mranftl.com/fonts/ibm-plex-mono
- https://gwfh.mranftl.com/fonts/lora
